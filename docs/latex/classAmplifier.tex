\doxysubsection{Detailed Description}
\mbox{\hyperlink{classAmplifier}{Amplifier}} Class. This takes two input pipes, one containing Audio and one containing a double scaling value. This will output to a pipe containing Audio which is the input audio scaled by the scaling value.
